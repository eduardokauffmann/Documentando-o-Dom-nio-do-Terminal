# 🧠 Guia de sobrevivência: Domínio do Terminal

Este repositório foi criado como parte da disciplina de **sistemas operacionais**, utilizando o simulador [Terminal Master]() para praticar comandos essenciais de Windows e Linux.

---

## Introdução 👍
A interface de linha de comando (CLI) é uma ferramenta poderosa que permite interagir com o núcleo do sistema operacional. Dominar esses comandos é fundamental para automação de tarefas, administração de servidores e desenvolvimento de software.

---

## 👨‍💻 Comandos aprendidos

Abaixo, os comandos que aprendi durante o treinamento gamificado:

| Comando | Função | Sistema | Exemplo de uso |
| :--- | :--- | :--- | :--- |
| pwd | Exibe o diretório atual | Linux/Unix | pwd |
| ls -la | Lista tudo (todos os arquivos e detalhes) | Linux/Unix | ls -la |
| cd X | Entrar em um diretório ou pasta | Ambos | cd Documentos |
| cat | Ler o conteúdo de um arquivo | Linux/Unix | cat arquivo.txt |
| grep | Buscar um texto dentro de arquivos | Linux/Unix | grep termo arquivo.txt |
| chmod | Alterar permissões de arquivos/pastas | Linux/Unix | chmod 777 script.sh |
| ps aux | Listar processos do sistema | Linux/Unix | ps aux |
| find | Buscar arquivos no sistema | Linux/Unix | find -name foto.png |
| \| (pipe) | Encadear a saída de um comando para outro | Ambos | cat log.txt \| grep erro |
| exit | Voltar ao menu ou sair do terminal | Ambos | exit |
| restart || :--- | :--- | :--- | :--- |
| clear | Limpa toda a tela do terminal | Ambos | clear |
| mkdir | Cria uma pasta nova | Ambos | mkdir nova_pasta |
| touch | Cria um arquivo vazio | Linux/Unix | touch notas.txt |
| rm | Apaga um arquivo | Linux/Unix | rm arquivo.txt |
| rmdir | Apaga uma pasta (se estiver vazia) | Ambos | rmdir pasta_vazia |
| cp | Copia um arquivo ou pasta | Ambos | cp arquivo.txt copia.txt |
| mv | Move ou renomeia um arquivo/pasta | Ambos | mv antigo.txt novo.txt |
| man | Abre o manual de instruções de um comando | Linux/Unix | man ls |
| echo | Exibe uma mensagem ou texto na tela | Ambos | echo Ola Mundo |
| head | Mostra as primeiras 10 linhas de um arquivo | Linux/Unix | head lista.txt |
| tail | Mostra as últimas 10 linhas de um arquivo | Linux/Unix | tail log.txt |
| history | Mostra a lista de comandos usados recentemente | Linux/Unix | history |
| whoami | Mostra o nome do usuário atual | Ambos | whoami |
| df -h | Mostra o espaço livre em disco (leitura fácil) | Linux/Unix | df -h |
| date | Exibe a data e a hora atual do sistema | Ambos | date |
